# 🚕 Uber Trip Analysis Dashboard | Power BI

## 📊 Overview
This project presents an interactive Power BI dashboard analyzing **100K+ Uber trips** to uncover insights into bookings, revenue, trip distance, and trip duration.  
The dashboard enables dynamic exploration of data through KPIs, filters, and drill-through capabilities.

---

## 🚀 Key Features
- 📈 Dynamic KPI tracking (Bookings, Revenue, Distance)
- 🧮 DAX-based measure selector for flexible analysis
- ⏱️ Time-based insights:
  - 10-minute pickup intervals
  - Hourly demand heatmaps
  - Weekday vs weekend trends
- 🔍 Drill-through reports for detailed analysis
- 🚗 Vehicle-type performance comparison
- 🎛️ Interactive filtering using slicers & bookmarks

---

## 🛠️ Tools & Technologies
- Power BI
- DAX (Data Analysis Expressions)
- Data Modeling
- Excel (Data Source)

---

## 📂 Repository Structure
- `Uber Trip Analysis 2.pbix` → Power BI Dashboard
- `Uber Trip Details.xlsx` → Main dataset
- `Location Table.xlsx` → Lookup dataset
- `Screenshot*.png` → Dashboard previews

---

## ▶️ How to Use
1. Download the `.pbix` file from this repository  
2. Open it using **Power BI Desktop**  
3. Load or refresh the dataset if required  
4. Use filters, slicers, and drill-through to explore insights
